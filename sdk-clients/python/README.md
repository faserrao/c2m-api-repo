# Python SDK (Placeholder)

## Current Python SDK

The active Python SDK is located at: `/sdk/python/`

To use the current SDK:
```bash
cd /sdk/python
pip install -e .
```

See the [Python SDK Documentation](/sdk/python/README.md) for complete details.
